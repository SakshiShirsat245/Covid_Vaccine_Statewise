# Covid_Vaccine_Statewise
Use the following covid_vaccine_statewise.csv dataset and perform following analytics on the given dataset
https://www.kaggle.com/sudalairajkumar/covid19-in-india?select=covid_vaccine_statewise.csv
a. Describe the dataset
b. Number of persons state wise vaccinated for first dose in India
c. Number of persons state wise vaccinated for second dose in India
d. Number of Males vaccinated d. Number of females vaccinated

The COVID Vaccine Data Analysis project focuses on analyzing state-wise vaccination data in India using the dataset covid_vaccine_statewise.csv. This project is an application of Data Science, where real-world data is processed to extract meaningful insights about vaccination progress.

The dataset contains detailed information about the number of vaccine doses administered across different states and union territories, including first dose, second dose, and gender-wise vaccination statistics (male and female). It also includes time-based records, allowing analysis of vaccination trends over a period.

In this project, data preprocessing techniques are applied to clean the dataset by handling missing values and removing inconsistencies. After preprocessing, the data is analyzed using tools like Python and Pandas to perform operations such as grouping, aggregation, and summarization.

The main objective of the analysis is to:

Determine state-wise vaccination counts for first and second doses
Calculate the total number of males and females vaccinated
Understand the distribution and progress of vaccination across India

This analysis helps in identifying vaccination patterns, comparing performance across states, and gaining insights into gender-wise participation in the vaccination drive.

Overall, the project demonstrates how data analytics techniques can be used to analyze large datasets and support decision-making in real-world scenarios such as public health monitoring.
